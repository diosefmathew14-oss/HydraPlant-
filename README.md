# 🌱 Hydra Plant – Smart Plant Monitoring System

Hydra Plant is an IoT-based smart plant monitoring system designed to help users monitor the health of their plants using sensor technology.

The system measures soil moisture, temperature, and humidity in real time using sensors connected to a microcontroller such as the ESP32. The collected data is sent to a monitoring dashboard where users can view plant conditions and receive alerts when watering is needed.

Unlike fully automated watering systems, Hydra Plant focuses on assisting plant owners by providing real-time information while still encouraging manual care and attention to plants.

## Features
• Real-time soil moisture monitoring  
• Temperature and humidity tracking  
• Smart notifications when plants need watering  
• Live dashboard with charts and gauges  
• IoT-based monitoring using sensors and microcontroller  

## Technologies Used
• HTML, CSS, JavaScript  
• IoT sensors (Soil Moisture Sensor, DHT Sensor)  
• ESP32 / Microcontroller  
• Cloud dashboard integration

## Purpose
This project was developed as a school IoT project to demonstrate how technology can help improve plant care and environmental monitoring.
