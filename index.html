Smart Plant Monitoring System

Hydra Plant is an IoT-based system designed to monitor soil moisture, temperature, and humidity in real time. Instead of automating watering, it helps users preserve the tradition of caring for plants while receiving smart alerts and monitoring data.

Key Features

Real-Time Monitoring

Track soil moisture continuously.

Temperature Monitoring

DHT sensor measures plant environment.

Humidity Tracking

Ensure optimal plant conditions.

Smart Alerts

Receive notifications when watering is needed.
